# 🎨 Portfolio Frontend (React)

This is the frontend of the Portfolio MERN application.
It allows users to add, edit, delete, and view projects using a simple UI.

---

## 🌐 Live Website

https://frontend-two-pi-10.vercel.app

---

## 🛠️ Tech Stack

* React.js
* HTML
* CSS
* Fetch API

---

## ✨ Features

* Add new projects
* Edit existing projects
* Delete projects
* View all projects

---

## ⚙️ Setup Instructions

Clone repository
git clone https://github.com/bhavyasaigatta-creator/frontend.git
cd frontend

Install dependencies
npm install

Run application
npm start

---
